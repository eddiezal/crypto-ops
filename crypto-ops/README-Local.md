# How to start CryptoOps (local + cloud)

1. Open PowerShell
2. Navigate to project folder:
    cd F:\CryptoOps
3. Run kickstart:


This will:
- Activate the venv
- Set Python path
- Check Python + pip
- Fetch local Coinbase prices
- Check Cloud Run service in sandbox/paper mode

cd → .\kickstart.ps1 → good to go.
